# Loss-type Playbook — natural next questions

Use to pick the right follow-up. Never run as a checklist.

## Wildunfall (deer / wild boar / fox)

Common, Teilkasko-covered. Required: Wildunfallbescheinigung from police or Förster. Ask: animal type if known, road/area, time of day, vehicle drivable, fluid leaks, airbags deployed, fur/blood evidence on vehicle. Skip: other-party fault, witnesses (rarely useful).

## Auffahrunfall (rear-end)

Liability usually with rear driver, but do not say so. Ask: who was front/rear, speeds, was front car braking/stopped, distance, road conditions, traffic light state, injuries (whiplash often delayed), Polizei-Vorgangsnummer.

## Parkschaden (parking lot / hit-and-run while parked)

Often Teilkasko if parked unattended and other party fled. Ask: where parked, when last seen intact, when found, note left? witnesses? CCTV nearby? Police report needed for Fahrerflucht claim.

## Glasbruch (windshield / glass)

Teilkasko-covered, no SF-Klasse impact. Ask: which glass (front, side, rear, sunroof), cause if known (stone, vandalism), still drivable, repair vs replacement preference, partner shop OK (Carglass / Junited).

## Fahrerflucht (hit-and-run, you are the victim)

Ask: did you see the other vehicle (color, make, partial plate), direction of travel, witnesses, CCTV, did you call police immediately. Police report is mandatory for coverage.

## Diebstahl (theft - whole vehicle)

Teilkasko. Ask: when last seen, where parked, all keys accounted for, original documents/keys location, anti-theft engaged, Polizei-Vorgangsnummer (mandatory).

## Teildiebstahl (parts / contents)

Ask: what was taken, signs of forced entry, where, when noticed, police report, value estimate, photos.

## Vandalismus

Vollkasko only. Ask: damage description, location, when discovered, witnesses, CCTV, police report, possible motive or known suspect.

## Sturm / Hagel / Naturereignis

Teilkasko. Ask: date and approximate time, location, wind/hail confirmed locally (use weather context tool if needed), shelter at time, damage type, photos.

## Brand

Teilkasko (non-self-caused) or Vollkasko. Ask: cause if known, location, fire department report, did fire spread, total loss?

## Internal-only nuances

- If caller fled the scene, do not call it Fahrerflucht to their face — document the facts neutrally.
- If alcohol or drugs are mentioned, document but do not flag coverage implications on the call.
- If telematics device is on the policy, do not mention it as a fault-finding tool.
